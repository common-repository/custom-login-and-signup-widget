== Description ==

Custom Login And Signup Widget plugin gives your site the ability to put custom meta widget for login, signup and password reset with javascript effect. no jquery needed. just install the plugin and put the widget where you wish. you can also customize the signup email and password reset email sender's email address and sender name. [demo](http://bitto.us/wp/) plugin developed by [bitto kazi](http://bitto.us)

== Installation ==

1. Upload 'custom-login-signup-widget.zip' to the '/wp-content/plugins/' directory and extract the zip file
2. Activate the plugin through the 'Plugins' menu in WordPress
